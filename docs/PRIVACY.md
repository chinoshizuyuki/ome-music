# Privacy

Ome Music is designed as a local-first music player.

## Local Data

The app may store:

- Local music metadata
- Playback history
- Lyrics cache
- Danmaku cache
- Music source settings
- Curator/API settings
- Small startup/session snapshots

## What Should Not Be Uploaded

- Local audio files
- Cookies or login tokens
- API keys
- SQLite databases
- Logs
- Diagnostics containing headers or account data
- Screenshots showing private accounts or playlists

## Network Sources

NetEase Cloud Music and Bilibili requests are made only for user-triggered source features such as search, playback, login, lyrics, video atmosphere, and danmaku.

Users are responsible for complying with third-party service terms.
